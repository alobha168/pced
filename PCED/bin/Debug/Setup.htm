本词典使用方便,查词速度快!
程序界面可以选择三种'繁体中文'，'简体中文'，'English'。
程序支持常用的Tahoma(unicode编码)字体和Sangayana字体
与VriRomanPali CN字体字母编码的自动转换和显示，查找的
结果可以直接复制为相应的字体。
使用时，请先根据您输入时所使用的字体来选择'字体设置'。
'输入模式'-'模糊输入模式'，可以输入英文字符，查对应词形
的巴利单词，在此模式下，您输入英文字符
'ara'，可以查到如下巴利单词：
Ara,【中】 轮辐。
āra,【阳】 针。
ārā,【无】 在远处，远离，遥远的。
如果不选中'模糊输入模式'，则输入'ara',只查到：
Ara,【中】 轮辐。
输入'ārā'，只查到：
ārā,【无】 在远处，远离，遥远的。
注意，在'模糊输入模式'下，您可以这样输入巴利文'ārā'，程序会自动转换为
'ara'，可查到相近词形：
Ara,【中】 轮辐。
āra,【阳】 针。
ārā,【阴】 锥子。
程序支持'复制取词'，当您在别的文档中选中了巴利单词，并'复制'的时候，本
程序会自动取得此单词，在输入要查找的'单词'后，可以直接按键盘上的'Enter'
按键，或点击'查词'按钮，即可查词。
如果要查找汉字的巴利语解释，可输入汉字，直接按键盘上的'Enter'按键，
或点击'查词'按钮，可查找相关内容以供参考。

怎样输入巴利音标字母：

    要输入巴利音标字母，在输入框里，只需要按住 Alt 键不放，然后按键盘上
对应的字母按键即可，根据不同的字体设置，可输入对应字体编码的巴利音标
字母，巴利音标与键盘英文字母对应关系如下：

Ā    Ī    Ū    Ṅ    Ñ   Ṭ    Ḍ    Ṇ    Ḷ    Ṃ   Ṁ    Ŋ
A    I    U    V    B    T    D    N    L    J    M    H
Alt + A = ā
Alt + Shift + A = Ā
